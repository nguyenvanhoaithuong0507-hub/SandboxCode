# 📖 Tổng quan

SandboxCode là một nền tảng AI-powered giúp tự động hóa
quy trình phát triển phần mềm.

## Mục đích

1. **Tự động hóa** việc viết code
2. **Kiểm thử** tự động mã nguồn
3. **Quản lý** các yêu cầu kéo (Pull Requests)
4. **Tối ưu hóa** quy trình phát triển

## Ai nên sử dụng?

- 👨‍💻 **Các lập trình viên** - Tăng tốc độ phát triển
- 🏢 **Các đội phát triển** - Cải thiện chất lượng code
- 🤖 **DevOps Engineers** - Tự động hóa CI/CD
