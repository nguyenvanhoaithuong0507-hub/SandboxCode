# 🤔 SandboxCodex là gì?

SandboxCodex là một **Kỹ sư phần mềm AI** với các khả năng:

## Các tính năng

### 1. 📊 Đánh giá Mã Thông Minh

- Phân tích cấu trúc code
- Kiểm tra best practices
- Đề xuất cải tiến

### 2. 🔧 Sửa lỗi Tự động

- Phát hiện bugs
- Đề xuất sửa chữa
- Xác nhận thay đổi

### 3. 🏃 Môi trường Chạy Sandbox

- Chạy code an toàn
- Kiểm thử tự động
- Biệt lập hoàn toàn

### 4. 🔀 Tích hợp Git

- Tạo commits
- Đẩy branches
- Tạo Pull Requests
