# SandboxCode Documentation Site

Đây là trang tài liệu Docusaurus cho dự án SandboxCode.

## Cài đặt

```bash
npm install
```

## Phát triển

```bash
npm start
```

Mở http://localhost:3000 để xem trang.

## Build

```bash
npm run build
```

## Deploy

```bash
npm run deploy
```

Deploy lên GitHub Pages tại:
https://nguyenvanhoaithuong0507-hub.github.io/SandboxCode/

## Cấu trúc Tệp

```
.
├── docs/                    # Tài liệu Markdown
│   ├── intro.md            # Trang chủ
│   ├── getting-started/    # Phần Bắt đầu
│   ├── guides/             # Hướng dẫn
│   └── reference/          # Tham khảo
├── docusaurus.config.js    # Cấu hình
├── sidebars.js             # Sidebar config
└── package.json            # Dependencies
```
