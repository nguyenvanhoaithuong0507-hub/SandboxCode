// PIPELINE VẬN HÀNH
Chu trình khép kín, tự phục hồi khi thất bại →
01

PLAN LẬP KẾ HOẠCH

Phân rã yêu cầu ngôn ngữ tự nhiên thành các tác vụ kỹ thuật cụ thể, xác định file cần đọc và phạm vi thay đổi.

02

CODE VIẾT CODE

Sinh và chỉnh sửa code trực tiếp trên nhánh làm việc, tuân theo convention và style hiện có của repository.

03

RUN THỰC THI

Chạy build, test và script trong môi trường sandbox cô lập, không chạm vào hạ tầng production.

04

OBSERVE QUAN SÁT

Đọc log, stack trace và kết quả test để xác định thay đổi có đạt yêu cầu hay còn lỗi cần xử lý.

05

PATCH VÁ LỖI

Tự động sửa và lặp lại chu trình đến khi test xanh, sau đó commit, push và mở Pull Request.

// BẮT ĐẦU TRONG 4 BƯỚC
Xem tài liệu đầy đủ →
01

Cài đặt CLI

Cài SandboxCodex CLI toàn cục qua trình quản lý gói bạn đang dùng.

$ pnpm add -g sandboxcodex
02

Kết nối GitHub

Xác thực OAuth và chọn repository muốn cấp quyền cho AI làm việc.

$ sandboxcodex login --provider github
03

Khởi tạo sandbox

Tạo môi trường chạy cô lập, tự động phát hiện framework và dependencies.

$ sandboxcodex init ./my-project
04

Giao việc đầu tiên

Mô tả tác vụ bằng ngôn ngữ tự nhiên, theo dõi pipeline và nhận Pull Request.

$ sandboxcodex run "sửa lỗi memory leak trong worker.ts"
