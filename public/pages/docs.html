SandboxCodex

SandboxCodex
"Ship code safely from an isolated sandbox." — your AI coding teammate

Cổng tự động hóa dành cho lập trình viên: chạy tác nhân AI, kiểm thử, review code và chuẩn bị pull request ngay trong một workspace an toàn.

Gửi yêu cầu, nhận bản vá, kiểm tra kết quả và tạo PR mà không phải rời khỏi luồng làm việc. SandboxCodex gom CLI, dashboard, templates và tích hợp GitHub vào một trải nghiệm tài liệu dễ đọc trên cả desktop lẫn mobile.

How it works

The sandbox is the single source of truth for runs, logs, generated patches, and pull request metadata.

Repository → Sandbox → AI Agent → Pull Request

Key capabilities

Automated reviewFind risky changes before they leave the workspace. Test orchestrationRun lint, unit, build, and custom checks consistently. Scoped secretsKeep tokens and environment variables controlled. PR generationPrepare titles, bodies, summaries, and test notes.

Quick start

  1. Install SandboxCodex

    bash
    npm install -g sandboxcodex

  2. Create a workspace

    bash
    sandboxcodex init my-project
cd my-project

  3. Open the dashboard

    bash
    sandboxcodex dashboard

Configuration (optional)

Config lives in sandbox.config.js. If you do nothing, SandboxCodex uses sensible defaults for local development, tests, and PR summaries.

js
module.exports = {
  projectName: 'my-project',
  sandbox: { runtime: 'nodejs', checks: ['lint', 'test', 'build'] },
  github: { token: process.env.GITHUB_TOKEN }
};
