# 💾 Cài đặt

## Cài đặt từ npm (Khuyến nghị)

```bash
npm install -g sandboxcode
```

## Cài đặt từ GitHub

```bash
npm install -g github:nguyenvanhoaithuong0507-hub/SandboxCode
```

## Cài đặt từ Source

```bash
git clone https://github.com/nguyenvanhoaithuong0507-hub/SandboxCode.git
cd SandboxCode
npm install
npm link
```

## Xác minh Cài đặt

```bash
sandboxcode --version
```

## Khắc phục sự cố

### ❌ Lỗi: "command not found: sandboxcode"

```bash
sudo npm install -g sandboxcode
```

### ❌ Lỗi: "Node.js version too old"

Cập nhật Node.js lên phiên bản 18.x trở lên.
